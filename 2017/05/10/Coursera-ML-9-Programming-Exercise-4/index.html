Neural Networks - Feed Forward and Cost Function

Sigmoid gradient

$$ g'(z) = g(z)(1 - g(z))$$

where $$ g(z) = \frac{1}{1+e^{-z}}$$

Cost Function

$$ J(\theta) = \frac{1}{m}\sum_{i=1}^{m}\sum_{k=1}^{K}\big[-y^{(i)}_{k}\, log\,(( h_\theta\,(x^{(i)}))_k)-(1-y^{(i)}_k)\,log\,(1-h_\theta(x^{(i)}))_k)\big]$$
