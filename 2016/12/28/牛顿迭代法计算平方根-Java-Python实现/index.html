    数学推导

    假设c为原数，t为c的根数。

    $$ t^2 \quad = \quad c$$
    $$2t^2 \quad = \quad c + \quad t^2$$
    $$2t \quad = \quad \frac{c}{t}+ \quad t$$
    $$t \quad = \quad \frac{\frac{c}{t} + t}{2.0}$$
