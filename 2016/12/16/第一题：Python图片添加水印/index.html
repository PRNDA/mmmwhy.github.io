代码解释

  • Image.composite(layer, im, layer) 解释一下Image.composite()的意思
    使用案例
    Image.composite(image1, image2, mask) => image
    使用遮罩（mask）作为alpha，通过在两个图像之间进行插值来创建一个新图像。遮罩图像的模式可以是“1”，“L”，或者“RGBA”。所有的图像的大小必须有相同。
